## 🚀 Featured Projects

### 🌐 [Network Traffic Analyzer](https://github.com/hayagrivan96-eng/network-traffic-analyzer)
*See Every Device. Understand Every Connection. Protect Every Packet.*
- A real-time network traffic analyzer built with a **React + TypeScript** dashboard and a **Python FastAPI** backend.
- Features device discovery (ARP/Ping), drag-and-drop force-directed network topology maps (ReactFlow), real-time bandwidth charts (Recharts), and hop-by-hop packet path traces.
- Includes a security auditor with port scanning logs and risk profiling.

### 📞 [Sheildcall AI](https://github.com/hayagrivan96-eng/sheildcallAi)
*Smart, AI-powered Call Screening & Privacy Protection*
- An intelligent call screening platform that identifies spam, transcribes call audio in real-time, and automates call forwarding.
- Protects communication privacy by leveraging AI to analyze call intent before connecting users.

### 🧪 [Mini Packet Tracer](https://github.com/hayagrivan96-eng/mini-packet-tracer)
*Lightweight Network Simulator & Design Tool*
- A lightweight desktop network simulation tool built with **Python and Tkinter**, inspired by Cisco Packet Tracer.
- Allows users to visually design network topologies, simulate packet routing, and inspect packets at a protocol level.

### 🛍️ **Drip Fashion** *(Private)*
